# Mindbrake — Legal

Páginas legales públicas de la app iOS Mindbrake. Servidas vía GitHub Pages.

## URLs

- Landing: https://gaelquinta.github.io/mindbrake-legal/
- Privacy Policy: https://gaelquinta.github.io/mindbrake-legal/privacy.html
- Terms of Service: https://gaelquinta.github.io/mindbrake-legal/terms.html

## Idiomas

Bilingüe ES/EN con switcher en la esquina superior derecha. Detección automática por `navigator.language` y override persistido en `localStorage`.

## Fuente

El texto debe mantenerse sincronizado con `BrainPass/Features/Settings/LegalView.swift` del repo Mindbrake. Si actualizas las vistas in-app, refleja los cambios aquí también.
